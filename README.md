# Market-X Backend

## Description
Market-X Backend is a RESTful API that powers the Market-X web application. It handles authentication, product management, and secure communication between the frontend and the database.

## Features
- User authentication using JWT
- Secure API with CORS and HTTP-only cookies
- Product CRUD operations
- Environment-based configuration
- Scalable Express architecture

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (JSON Web Token)

## Architecture Pattern
 - MVC (Model–View–Controller)

## Frontend Repository
 - The frontend is built with React.js and deployed separately.
